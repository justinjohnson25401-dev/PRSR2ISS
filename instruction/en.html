Extension Usage Instructions (English)

  1. Open the 2GIS Maps website.
    Go to 2gis.com.
  2. Search for the places you need.
    Enter a query (for example, “cafe Moscow”). All visible places shown in the list or on the map will be automatically saved into the table. The counter on the extension icon shows how many places have been saved.
  3. Download the table.
    Once you’ve gathered enough places, click the extension icon and select “Download table.” All collected businesses will be combined into a single file and saved to your computer.
  4. Reset the table.
    To start a new collection, click “Reset” and open new businesses on the map.
